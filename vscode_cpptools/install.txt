1. 下载和安装cursor的老版本，比如1.2.1
2. 下载老版本的vscode的cpp_tools插件
    获取插件信息​
    ​发布者（Publisher）​: ms-vscode
    ​插件名（Extension Name）​: cpptools
    ​版本号（Version）​: 在 官方页面 查看最新版本号（如 2.0.0）
    ​拼接下载链接​
    使用格式：
    url
    复制
    https://marketplace.visualstudio.com/_apis/public/gallery/publishers/ms-vscode/vsextensions/cpptools/[版本号]/vspackage?targetPlatform=[平台]
    ​示例​：
    Linux x64:
    url
    复制
    https://marketplace.visualstudio.com/_apis/public/gallery/publishers/ms-vscode/vsextensions/cpptools/2.0.0/vspackage?targetPlatform=linux-x64
    Windows 64-bit:
    url
    复制
    https://marketplace.visualstudio.com/_apis/public/gallery/publishers/ms-vscode/vsextensions/cpptools/2.0.0/vspackage?targetPlatform=win32-x64
    macOS Apple Silicon:
    url
    复制
    https://marketplace.visualstudio.com/_apis/public/gallery/publishers/ms-vscode/vsextensions/cpptools/2.0.0/vspackage?targetPlatform=darwin-arm64
    ✅ ​注​：将 [版本号] 替换为实际版本，[平台] 替换为你的操作系统类型
    
    https://marketplace.visualstudio.com/_apis/public/gallery/publishers/ms-vscode/vsextensions/cpptools/1.23.6/vspackage?targetPlatform=linux-x64

   比如Linux的1.23.5版本：https://marketplace.visualstudio.com/_apis/public/gallery/publishers/ms-vscode/vsextensions/cpptools/1.23.5/vspackage?targetPlatform=linux-x64


3. 如果是远端开发，上传vsix文件到远端服务器
4. 手工安装
   4.1 ctrl + shift + p
   4.2 Extensions: Install from VSIX...

注意： 
1. 要禁用Anysphere的cpptools, 这个跟vscode的cpptools有冲突
2. .vscode/c_cpp_properties.json是否需要配置，1.2.1的不需要，其他版本可能需要配置
